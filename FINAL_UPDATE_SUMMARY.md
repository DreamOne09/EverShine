## 📊 最終配置總覽

### 星空系統配置

```javascript
// 靜態星星（滿天星）
staticStars: [
    { count: 200, animationDuration: '100s' },  // 第一層
    { count: 180, animationDuration: '125s' },  // 第二層
    { count: 160, animationDuration: '175s' },  // 第三層
    { count: 140, animationDuration: '200s' },  // 第四層
]
// 每層都有 ::after 偽元素再加一倍
// 實際顯示: ~1360 顆靜態星星

// 移動星星
movingStars: [
    { count: 120, size: '2px', duration: '125s' },
    { count: 100, size: '3px', duration: '175s' }
]
// 實際顯示: 220 顆移動星星

// 流星（減少數量）
shootingStars: {
    minCount: 2,      // 最少 2 顆
    maxCount: 3,      // 最多 3 顆
    minSpeed: 10,     // 速度加快
    maxSpeed: 18
}
```

### 各區塊 z-index 層級

```css
/* 從後到前的層級順序 */
body::after (nebula)              z-index: -5   (最底層)
Canvas 星空 (#starrySkyCanvas)    z-index: -3
CSS 星空容器                       z-index: -1
靜態/移動星星層                    z-index: 1    (在容器內)
流星 (.shooting-star)             z-index: 2    (在星空之上)
所有內容區塊 (section)             z-index: 10   (確保在背景之上)
導航欄 (.navbar)                  z-index: 1000 (最上層)
```
